# Overview

MedBook is booking project that enables patients to search on available doctors and make an appointment for them

# Get Started

1. You need to follow the instructions in `NOTE.md` file to setup **.env** files for the projects
2. Instal dependencies

```bash
pnpm install
```

3. Generate Priam Types

```bash
pnpm run --filter @repo/db prisma:generate
```

4. Build `@repo/db` project

```bash
pnpm run --filter @repo/db build
```

5. Build `@repo/shared` project

```bash
pnpm run --filter @repo/shared build
```

6. Run Backend project

```bash
pnpm run --filter backend dev
```

7. Run Frontend project

```bash
pnpm run --filter frontend dev
```

8.  (Optional ) Run Prisma Studio to visualize data in Postgres db

```bash
pnpm run --filter @repo/db prisma:studio
```

# Technologies

I used `MonoRepo` architecture to share some types and schemas and manage dependencies between projects

### Projects

- Backend: NestJs
- Frontend: Next.js
- DB: ORM Prisma & Generated Types
- Shraed: Zod Schemas

### Backend Stack

- Framework: NestJs
- Security: JWT - bcrypt - rate limiting - csurf - helmet
- Validation: nest-zod - Zod
- Db Driver: PgPrisma

### Frontend Stack

- Framework: Next.js
- UI Kit: Shadcn UI
- Css Framework: Tailwindcss
- Icons: Lucid React
- Toast: Sonner
- Data Fetching: Axio - React-Query
- Search Params States: Nuqs
- Animation: Motion
- Validation: React Hook Form - Zod
- Maps: Leaflet

### Db Project Stack

- ORM: Prisma
- Types: Generated by prisma cli

### Shared Project

- Schemas: Zod
- Dto: Normal classes and interfaces
